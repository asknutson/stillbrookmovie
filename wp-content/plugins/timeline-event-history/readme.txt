== Installation ==

1. Upload the entire `timeline-event-history` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3.Create gallery and copy its shotcode. 
4. Now paste its shortcode to any page/post/widget.
